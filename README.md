# 📈 Smart SIP Calculator (Vanilla JS)

A modern SIP (Systematic Investment Plan) calculator with **real-time calculation, step-up investment support, and visual growth chart**.

Built using pure HTML, CSS, and JavaScript — no frameworks, no dependencies.

---

## 🚀 Live Demo

👉 https://yuvronixstudio.github.io/sip-calculator/

---

## ✨ Features

- 📊 Real-time SIP calculation (instant updates)
- 📈 Visual growth chart (Invested vs Total Value)
- 🔁 Step-up SIP support (increase investment yearly)
- 💡 Accurate monthly compounding logic
- 📱 Fully responsive design
- ⚡ Lightweight and fast (no libraries)
- 🌐 Works offline

---

## 🧮 What It Calculates

- Total invested amount  
- Estimated returns  
- Final maturity value  

Simulates realistic SIP growth with optional yearly step-up.

## ⚙️ How to Use

1. Clone or download the repository
2. Open `index.html` in your browser
3. Enter:
   - Monthly investment
   - Expected return
   - Duration
   - Step-up %
4. View results instantly
